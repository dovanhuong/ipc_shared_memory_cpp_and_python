# ipc_shared_memory_cpp_and_python
The open source for using the shared memory between Cpp and Python script

Instruction to run the script:
1. Compile source Cpp
  `$ ./compile.sh`
3. Run Cpp binary
  `$ ./shared_mem`
5. Run Python script
   `$ python shared_mem.py`
